# miniOSL

Python interface to [OSL (open shogi library)](https://gps.tanaka.ecc.u-tokyo.ac.jp/gpsshogi/index.php?GPSshogi), refurbished and enhanced by C++20 with pybind11 for cooperation with numpy and pytorch.
